## 📌 Project Overview
This project is my **Final Year Project (PFE)** for the Bachelor's degree in Computer Science at **University Saad Dahleb - Blida 1**.

The application is designed to automate and optimize the management of university timetables. It solves the problem of manual scheduling by automatically handling teacher availability, room assignments, and student group constraints.

## 🚀 Key Features
- **Automated Scheduling:** Algorithms to generate timetables based on specific constraints.
- **Conflict Management:** Automatic detection and resolution of teacher/room overlaps.
- **Real-time Updates:** Instant synchronization for teachers and students.
- **User Dashboards:** Specialized views for Administrators, Faculty, and Students.

## 🛠️ Tech Stack
- **Backend:** Node.js, Express
- **Frontend:** HTML5, CSS3, JavaScript
- **Database:** MySQL / MongoDB
- **Version Control:** Git & GitHub
